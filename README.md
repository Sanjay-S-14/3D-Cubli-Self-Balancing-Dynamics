# 3D Cubli: Edge Stabilization and Dynamic Control

## 📌 Overview
This project focuses on the dynamic modeling and control system simulation of a 3D Cubli. The current phase successfully achieves robust edge balancing using advanced control theory. This serves as a critical intermediate milestone in managing multi-body dynamics before scaling up to full 3D spatial vertex (corner) balancing.

## ⚙️ Current Control Architecture: Edge Balancing

Balancing the cube along a 1D edge requires rejecting internal parameter variations and external disturbances.

* **Sliding Mode Control (SMC):** Implemented to achieve robust edge & corner stabilization. SMC drives the system states to a sliding surface and maintains them there, providing high accuracy despite non-linear system dynamics.
* **Quaternion-Based Kinematics:** Transitioning spatial dynamics to quaternions and rigid transforms to avoid gimbal lock during full 3D rotation.
* **Gyroscopic Decoupling:** Upgrading the control law to actively address and mitigate the highly non-linear gyroscopic coupling effects generated by all three internal reaction wheels spinning simultaneously.
* **Multi-Axis SMC:** Expanding the Sliding Mode Controller to handle simultaneous, coupled multi-axis stabilization.
* * **Numerical Stability:** Employed stiff solvers (ode15s VSVO solver) to maintain precision throughout the dynamic simulation, preventing integration failures caused by rapid reaction wheel dynamics.

## 🚀 Current Focus & Future Scope (Including Imitation Real Life Sensor data)

### 1. Transition to Physical Hardware & Real-Life Sensor Data
*   **Hardware Migration:** Transition the control loops from ideal simulation state-space feedback to physical Inertial Measurement Unit (IMU) data (e.g., MPU6050 or MPU9250).
*   **Hardware Constraints:** Calibrate the system to dynamically compensate for physical non-idealities, including sensor bias, temperature-induced drift, axial misalignment, and hardware sampling latency.
*   **Actuator Limitations:** Factoring in real-world motor driver dead-zones, voltage drops, and maximum torque saturation limits under continuous operation.

### 2. Implementation of Advanced Filtering Techniques
*   **Sensor Fusion:** Integrate a dedicated state-estimation filter—such as a Complementary Filter or a Kalman Filter—to accurately combine high-frequency gyroscope data with stable, low-frequency accelerometer data.
*   **Noise & Chattering Mitigation:** Filter out structural high-frequency vibrations caused by the high-speed reaction wheels and jumping mechanism impacts.
*   **Controller Optimization:** Preventing raw sensor noise from feeding directly into the Sliding Mode Controller (SMC) switching function, which minimizes high-frequency motor chattering, reduces heat generation, and conserves battery power.
